# Quiet Resolve (Calm Micro App)

Installable calming micro-app (PWA) with:
- Soft blush, sage, and pastel-purple animated background cycle
- Breathing orb with adjustable pace
- Tap-to-release petals
- Calming prompt generator

## Publish to GitHub (no terminal required)
1. Go to [https://github.com/new](https://github.com/new).
2. Create a new repository, for example: `quiet-resolve-calm-app`.
3. Open the repo page and click `uploading an existing file`.
4. Drag all files from this folder into GitHub:
   `C:\Users\amule\OneDrive\Documents\Brands\Everyday Modern... But Why\Trigger Calm\The Architect of Quiet Resolve Build Your Calm on Purpose\Test\calm-micro-app`
5. Commit to `main`.
6. In GitHub repo, open `Settings > Pages`.
7. Under `Build and deployment`, set `Source` to `GitHub Actions`.
8. Wait for the `Deploy Calm App to Pages` workflow to finish.
9. Your app URL will be:
   `https://<your-github-username>.github.io/<your-repo-name>/`

## Install as an app from GitHub Pages
- Desktop Chrome/Edge: browser menu -> `Install app`
- Android Chrome: menu -> `Install app` or `Add to Home screen`
- iPhone/iPad Safari: `Share` -> `Add to Home Screen`
